Как можно управлять?

  1. Кнопками на панели управления
  2. Кликая по миниатюрам на навигационной панеле
  3. Горячими клавишами
  4. Колесом мыши
  5. Программно, используя методы презентации
  6. Включить autoplay с самого начала и наслаждаться

Из навигации есть что?

  1. Навигационная панель с миниатюрами слайдов [ optional ]
  2. Номера слайдов и их состояние на навигационной панеле [ optional ]
  3. Подписи к слайдам на навигационной панеле [ optional ]
  4. Номер текущего слайда [ optional ]
  5. Прогрессбар презентации [ optional ]

А как же анимация?

Для презентации можно задать эффект смены слайдов!

А если меня не устроит решение из коробки?

Кому известны слова deferred и promice, сможет без проблем добавить и заюзать свой любимый кастомный эффект!
